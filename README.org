* Directions:

- Run the makefile
- Start the game server: `./gs`
- Start two instances of the client, one for each player: `./cl`
- Play the game!

*  Protocol

|-----+-------------|
| Tag | For         |
|-----+-------------|
| m   | match       |
| b   | board       |
| p   | play        |
| t   | turn        |
| i   | information |
|-----+-------------|


|-----------+-----------+-----------------------------------|
| Client    | Direction | Server                            |
|-----------+-----------+-----------------------------------|
| join      | -->       |                                   |
|           | <--       | [ok or bad] m# b#-#-#-#-#-#-#-#-# |
| leave m#  | -->       |                                   |
|           | <--       | [ok or bad] m#                    |
| move p#-# |           |                                   |
|-----------+-----------+-----------------------------------|
